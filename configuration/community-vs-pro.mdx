Mega Brain is available in two editions — Community (L1) and Pro (L2). Both share the same CLI and setup process; Pro unlocks the full knowledge pipeline and advanced agent capabilities. Start with Community to learn the system, then upgrade to Pro when you're ready to operationalize expert materials at scale.

## Feature Comparison

| Feature | Community (L1) | Pro (L2) |
|---|---|---|
| CLI and setup wizard | ✓ | ✓ |
| Core engine and templates | ✓ | ✓ |
| Skills and hooks | ✓ | ✓ |
| Agent templates and examples | ✓ | ✓ |
| Populated knowledge base | — | ✓ |
| Mind clone agents | — | ✓ |
| Pipeline processing | — | ✓ |
| Council / Conclave | — | ✓ |

## Community Edition

Community is a great way to get started and learn the Mega Brain system without any commitment. You get full access to the CLI, the core engine, and everything you need to understand how knowledge pipelines work before you invest in your own materials.

With the Community edition you can:

- Install and configure Mega Brain using the same setup wizard as Pro
- Use the CLI and all slash commands, including JARVIS orchestration
- Create cargo agents from the included agent templates and examples
- Access the full suite of skills and automation hooks
- Experiment with the core engine and explore how DNA extraction and playbook generation work

Community gives you a working system you can learn from and build intuition with, even before you bring your own expert materials into the pipeline.

## Pro Edition

Pro unlocks the complete Mega Brain knowledge pipeline and every advanced capability built on top of it. If your goal is to capture expert knowledge, create grounded AI agents, and run structured multi-agent deliberations, Pro is where that becomes possible.

With the Pro edition you unlock:

- **Full pipeline processing** — run the complete Ingest → Process → Extract DNA → Create Agents → Deliberate flow on your own materials
- **Mind clone agents** — build AI agents grounded in your ingested expert materials, with full provenance from every response back to its source
- **A populated knowledge base** — start from a pre-built foundation rather than an empty system
- **Conclave multi-agent deliberation sessions** — convene your Council of mind clone agents to deliberate on complex questions and synthesize structured outputs
- **Full provenance tracking** — trace every claim in every output directly back to the source document and passage it came from
