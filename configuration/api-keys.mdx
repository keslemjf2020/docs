Mega Brain uses API keys to access AI services. The only required key is your OpenAI API key; all others are optional and prompted during setup. You can supply keys through the interactive setup wizard or by editing the `.env` file directly.

## Required Keys

Mega Brain will not start without a valid `OPENAI_API_KEY`. Every language model operation — extraction, synthesis, agent reasoning, and output generation — runs through OpenAI.

<ParamField path="OPENAI_API_KEY" type="string" required>
  Your OpenAI secret key. Obtain one from the [OpenAI API keys dashboard](https://platform.openai.com/api-keys). Keys begin with `sk-`.
</ParamField>

Add the key to your `.env` file in the following format:

```bash .env
OPENAI_API_KEY=sk-...
```

## Optional Keys

During setup, Mega Brain may prompt for additional API keys depending on the integrations and features you choose to enable. These keys are entirely optional — Mega Brain runs without them. Press **Enter** at any optional prompt to skip it and continue setup.

## Setting Up Keys

The fastest way to configure your keys is through the interactive setup wizard.

<Steps>
  <Step title="Run the setup wizard">
    Execute the setup command in your project directory:

    ```bash
    npx mega-brain-ai setup
    ```

    The wizard walks you through every key it needs, one prompt at a time.
  </Step>
  <Step title="Enter your OpenAI API key">
    When the wizard prompts for `OPENAI_API_KEY`, paste your key and press **Enter**. This is the only key you must provide.
  </Step>
  <Step title="Skip optional keys">
    For every optional key prompt, press **Enter** without typing a value to skip it. You can always re-run `npx mega-brain-ai setup` later to add optional keys.
  </Step>
  <Step title="Confirm the .env file is created">
    After setup completes, a `.env` file appears in your project directory containing all the keys you entered. Mega Brain reads this file automatically on every run.
  </Step>
</Steps>

## Manually Editing .env

You can also create or edit the `.env` file directly without running the wizard. Open (or create) `.env` in your project root and add key-value pairs, one per line:

```bash .env
OPENAI_API_KEY=sk-your-key-here
```

Save the file and restart Mega Brain for changes to take effect.

<Warning>
  Never commit your `.env` file to version control. It contains secrets that grant access to your AI accounts. Add `.env` to your `.gitignore` immediately after setup:

  ```bash
  echo ".env" >> .gitignore
  ```
</Warning>

<Note>
  If Mega Brain cannot find a `.env` file when you first run a command, it automatically triggers the setup wizard so you can enter your keys before anything else runs. You will never be left without a clear prompt to provide credentials.
</Note>

For installation steps that precede key configuration, see the [installation guide](installation).
