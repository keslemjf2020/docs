Mega Brain is a Claude Code-powered knowledge system that ingests expert materials — videos, PDFs, transcriptions, podcasts, and training courses — and transforms them into structured, queryable intelligence. Instead of letting valuable expertise sit in scattered files and forgotten notes, you get actionable playbooks, 5-layer DNA schemas, and AI agents that reason like the experts whose work you've studied.

## Get up and running

Follow these four steps to go from a fresh install to your first structured knowledge artifact.

<Steps>
  <Step title="Install Mega Brain">
    Install the package globally via npm or run it directly with npx, then let the setup wizard handle dependencies.

    ```bash
    npm install -g mega-brain-ai
    # or
    npx mega-brain-ai setup
    ```
  </Step>
  <Step title="Configure your API keys">
    The setup wizard prompts you for your keys and writes them to `.env`. Only `OPENAI_API_KEY` is required — it powers video and audio transcription via Whisper.

    ```bash
    # Minimal .env
    OPENAI_API_KEY=sk-your-key-here
    ```
  </Step>
  <Step title="Ingest your first material">
    Open your Mega Brain project in Claude Code and drop in a YouTube URL or a local PDF path.

    ```bash
    /ingest https://www.youtube.com/watch?v=YOUR_VIDEO_ID
    ```
  </Step>
  <Step title="Process, extract, and query">
    Run the JARVIS pipeline, extract the expert's cognitive DNA, and start asking questions of a mind-clone agent trained on their actual material.

    ```bash
    /process-jarvis
    /extract-dna expert-name
    /ask expert-name "What's your framework for pricing a high-ticket offer?"
    ```
  </Step>
</Steps>
