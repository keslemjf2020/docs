A Mind Clone is an AI agent trained on a specific expert's actual materials — their videos, books, courses, and interviews — rather than on general knowledge. Instead of summarizing what an expert might say, a Mind Clone reasons from their documented philosophies, mental models, and frameworks, and backs every response with citations to the source material it was built from. In this guide you'll go from raw ingested materials to a fully functioning Mind Clone Agent.

<Note>
  Mind Clone Agents are a **Pro edition** feature. You need an active Mega Brain Pro subscription to run `/extract-dna` on expert-attributed knowledge or to create Mind Clone type agents. Community edition users can create [Cargo Agents](/concepts/agents) instead. See [Community vs Pro](/configuration/community-vs-pro) for details.
</Note>

## Prerequisites

Before you create a Mind Clone, make sure you have:

- **Ingested at least one expert's materials** using `/ingest`. The more sources you've ingested, the richer the resulting agent. See [Ingest Expert Materials](/guides/ingesting-materials) if you haven't done this yet.
- **A Mega Brain Pro project** initialized in your current working directory.
- **Claude Code** open and pointed at your project root.

## Building Your Mind Clone

<Steps>
  <Step title="Ingest the expert's materials">
    If you haven't already, run `/ingest` for every piece of source material you want the Mind Clone to draw from. YouTube videos, PDFs, audio recordings, and course materials all contribute to the agent's cognitive profile.

    ```bash
    /ingest
    ```

    Repeat this step for each source. A Mind Clone built on a single interview will be narrower than one built on an expert's full published body of work. See [Ingest Expert Materials](/guides/ingesting-materials) for the complete ingestion walkthrough.
  </Step>
  <Step title="Run DNA extraction">
    Once all materials are ingested, run `/extract-dna` and specify which expert or material set to extract from. JARVIS will analyze every ingested source and distill the expert's thinking into the 5-Layer DNA schema.

    ```bash
    /extract-dna
    ```

    When prompted, specify the target:

    ```
    Extract from: Alex Hormozi
    ```

    JARVIS will identify and structure:

    - **Philosophies** — Core beliefs and worldviews
    - **Mental Models** — Recurring thinking patterns and cognitive lenses
    - **Heuristics** — Rules of thumb and decision shortcuts
    - **Frameworks** — Structured approaches to categories of problems
    - **Methodologies** — Step-by-step processes and repeatable systems
  </Step>
  <Step title="Review the DNA schema">
    JARVIS outputs a structured cognitive profile once extraction completes. Read through it carefully before proceeding. Verify that the extracted philosophies and frameworks match what you know about the expert from the source materials.

    ```
    ✓ DNA extraction complete: Alex Hormozi
      → 5 philosophies extracted
      → 12 mental models identified
      → 8 heuristics documented
      → 6 frameworks structured
      → 3 methodologies mapped
      → Schema saved to ./dna/alex-hormozi.json
    ```

    If something looks missing or misattributed, ingest additional materials and re-run `/extract-dna` before creating the agent.
  </Step>
  <Step title="Create the Mind Clone agent">
    With the DNA schema confirmed, run `/create-agent` to instantiate the Mind Clone.

    ```bash
    /create-agent
    ```

    When prompted for agent type, select **Mind Clone**:

    ```
    Agent type: Mind Clone
    Based on DNA schema: alex-hormozi
    Agent name: Alex Hormozi Mind Clone
    ```

    JARVIS wires the agent to the extracted DNA and all underlying source materials, establishing the provenance links that power evidence-based responses.
  </Step>
  <Step title="Test the agent">
    Ask the Mind Clone a question relevant to the expert's domain. Observe how it structures its response — a well-built Mind Clone reasons through the expert's documented frameworks and cites the specific source material behind each claim.

    ```
    You: How should I price a new service offer?

    Alex Hormozi Mind Clone:
    Based on the "Value Equation" framework (source: $100M Offers, Chapter 4),
    price should reflect perceived value, not cost. The four levers are...
    [citation: youtube/hormozi-pricing-talk.mp4 @ 14:32]
    ```

    If the citations are thin or the reasoning doesn't reflect the expert's style, ingest more diverse material and re-extract the DNA.
  </Step>
</Steps>

## What Makes a Good Mind Clone

Not all Mind Clones are equal. The quality of the agent scales directly with the quality and volume of its source material.

<CardGroup cols={3}>
  <Card title="More Material = Richer Agent" icon="layer-group">
    Each additional ingested source adds new concepts, examples, and nuance to the DNA schema. A Mind Clone built from 20 sources will outperform one built from 2.
  </Card>
  <Card title="Diverse Format Types" icon="shapes">
    Combining videos, PDFs, and courses builds a more complete cognitive profile than any single format. Each medium captures different facets of how an expert thinks and communicates.
  </Card>
  <Card title="Evidence Citations" icon="link">
    Every response the Mind Clone generates is traceable to a specific moment in a specific source. This grounding prevents hallucination and lets you verify claims directly against the original material.
  </Card>
</CardGroup>
