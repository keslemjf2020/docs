Mega Brain is distributed across two primary editions: Community (L1), the open-source foundation available as a public npm package, and Pro (L2), which layers in a populated knowledge base, pre-built mind clone agents, and full pipeline processing. Both editions share the same core engine, hooks, skills, and CLI tools — Pro simply adds premium content on top of that foundation.

## Distribution Layers

Mega Brain uses a three-layer architecture. Your edition determines which layers are included.

| Layer | Name | Content | Distribution |
|---|---|---|---|
| **L1** | Community | Core engine, templates, hooks, skills, CLI | npm package (public) |
| **L2** | Pro | Populated knowledge base, mind clones, pipeline | Premium (tracked) |
| **L3** | Personal | Your materials, sessions, environment config | Local only (gitignored) |

<Info>
Layer 3 is always local-only and gitignored regardless of your edition. Your personal data — inbox files, knowledge base, session logs, and `.env` — never leaves your machine.
</Info>

## Prerequisites

Both editions share the same system requirements.

| Requirement | Version | Notes |
|---|---|---|
| [Claude Code](https://claude.ai/claude-code) | Max or Pro plan | Core runtime |
| [Node.js](https://nodejs.org) | >= 18.0.0 | CLI and tooling |
| [Python](https://python.org) | >= 3.10 | Intelligence scripts |

### API Keys

| Key | Purpose | Required? |
|---|---|---|
| `OPENAI_API_KEY` | Whisper transcription | **Yes** |
| `VOYAGE_API_KEY` | Semantic embeddings (RAG) | Recommended |
| `GOOGLE_CLIENT_ID` | Google Drive import | Optional |

Run `/setup` in Claude Code to configure keys interactively. All keys are stored in `.env`, which is gitignored.

## Upgrading from Community to Pro

Pro Edition builds directly on top of Community, so upgrading is seamless and requires no migration.

- All your L3 (Personal) data remains completely intact
- Your custom configurations and agent definitions are preserved
- Your `.env` settings carry over without modification
- Community customizations work alongside Pro features

<Note>
To access Pro Edition, contact **@thiagofinch** on GitHub for access and licensing information. Your personal layer (L3) data is never affected by the upgrade.
</Note>
